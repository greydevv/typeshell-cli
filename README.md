# typeshell-cli

## Description
typeshell-cli is a Python command-line-interface tool that was written as a proof-of-concept of [pytyper](https://github.com/greysonDEV/pytyper), a Python typing test package.

## Installation
```
pip install typeshell
```

## Usage

```
$ typeshell
```
